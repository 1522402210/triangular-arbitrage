# triangular-arbitrage
数字货币-三角套利机器人

## 配置
1、config/default.toml.org 改为 config/default.toml

2、config/default.toml文件中配置修改，例如：币安apikey

## 启动步骤

```js
npm install
npm start
```

## 疑难解答

Q：toml配置如何改为json配置?

A：可以把toml后缀改成json,然后通过[这个地址](https://toml-to-json.matiaskorhonen.fi/)，把toml格式配置转换成json格式。
